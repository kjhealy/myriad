# theme_myriad_semi

A \[ggplot2\] theme using semibold variants of Adobe Myriad Pro

## Usage

``` r
theme_myriad_semi(
  base_size = 12,
  base_family = "Myriad Pro SemiCondensed",
  title_family = "Myriad Pro Semibold SemiCondensed",
  base_line_size = base_size/24,
  base_rect_size = base_size/24
)
```

## Arguments

- base_family, title_family, base_size, base_line_size, base_rect_size:

  base font family and size

## Details

You should \[import_myriad_semi\]() first and also install the fonts on
your system before trying to use this theme.
