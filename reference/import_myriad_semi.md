# Import Myriad SemiCondensed only

This will work for devices that use \`systemfonts\` to process fonts.
For PDFs, use showtext::showtext_auto() beforehand

## Usage

``` r
import_myriad_semi()
```

## Value

Makes "Myriad SemiCondensed" available for use
